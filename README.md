# Modern Portfolio Website - Dang Hoang Trung Kien

A high-end, responsive portfolio website built with HTML, CSS, and Vanilla JavaScript. Designed for a premium look with glassmorphism, smooth animations, and a dark theme.

## 🚀 Features

- **Premium UI/UX**: Dark mode aesthetic with glassmorphism and subtle gradients.
- **Dynamic Animations**: 
  - Hero area with morphing avatar background.
  - Scroll-reveal effects for all sections.
  - Smooth anchor scrolling.
  - Interactive skill tags and project cards.
- **Detailed Resumé**: Full integration of professional experience, education, and technical skills.
- **Fully Responsive**: Optimized for seamless viewing on mobile, tablet, and desktop devices.
- **Clean Code**: Semantic HTML5 and modern CSS3 (Flexbox/Grid architecture).

## 🛠️ Built With

- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3**: Custom design system using variables and `@keyframes` animations.
- **JavaScript**: Vanilla JS for scroll logic and intersection observer animations.
- **Google Fonts**: Custom typography using 'Outfit'.

## 📁 Project Structure

```text
/
├── index.html        # Main structure & content
├── style.css         # Design system & animations
├── script.js         # Interactive logic & scroll effects
├── avatar_kien.JPEG  # Professional avatar image
└── README.md         # Project documentation
```
